# ES6 Prototype Contact App

## Installation
```
Run npm install or yarn from the root directory.
```
To fire up the dev server on `http://localhost:8080/`, type `npm run start` or `yarn start` from the app's root directory.


### Folder Structure


```shell
app/  # → Root folder
├── src/
  ├── lib/
      └── components
      app.js
  └── scss
      index.html
      index.js
```
